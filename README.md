# 🎵 GROOVE — Web Music Player

> A sleek, dark-themed web-based music player with vinyl disc animation, playlist management, and full audio playback controls — built with pure HTML, CSS & JavaScript.

---

## 🔗 Live Demo

👉 **[Click here to view the live site](https://Aakritijain30.github.io/groove-music-player)**

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎡 Vinyl Animation | Spinning disc with tonearm that moves on play/pause |
| ▶️ Playback Controls | Play, Pause, Next, Previous |
| ⏩ Seek Bar | Click anywhere on progress bar to jump to that position |
| 🔊 Volume Control | Smooth volume slider with percentage display |
| 🔀 Shuffle | Randomly pick next song |
| 🔁 Repeat | Loop current song |
| 🎨 Per-Song Colors | Each song has its own accent color on vinyl label |
| 📻 EQ Animation | Animated equalizer bars on the active track |
| 📱 Responsive | Works on mobile too |

---

## 🛠️ Tech Stack

- **HTML5** — Structure & audio element
- **CSS3** — Animations, grid layout, custom scrollbar
- **Vanilla JavaScript** — No frameworks, no libraries
- **Google Fonts** — Bebas Neue, Space Mono, DM Sans

# 🎵 GROOVE — Music Player

A sleek, dark-themed web music player with vinyl disc animation, playlist, search bar, and full playback controls. Built with pure HTML, CSS & JavaScript — no frameworks!

## ✨ Features

| Feature | Description |
|---|---|
| 🎨 Vinyl Disc Animation | Spinning vinyl with tonearm that moves on play/pause |
| ▶️ Playback Controls | Play, Pause, Next, Previous |
| 🔍 Search Bar | Search songs by title or artist name in real time |
| 🔀 Shuffle | Play songs in random order |
| 🔁 Repeat | Loop current song |
| 🔊 Volume Control | Adjustable volume slider |
| ⏱️ Seek Bar | Click anywhere on progress bar to jump |
| 📋 Playlist | Interactive playlist with animated EQ bars |
| 🎨 Per-song Colors | Each song has its own accent color theme |

---

## 🛠️ Tech Stack

- **HTML5** — Structure & audio element
- **CSS3** — Animations, grid layout, custom scrollbar
- **Vanilla JavaScript** — No frameworks, no libraries
- **Google Fonts** — Bebas Neue, Space Mono, DM Sans
